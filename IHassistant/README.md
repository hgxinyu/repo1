# IHassistant

放置奇兵游戏助理知识库与推荐系统。

这个项目会逐步收集并整理游戏机制、英雄技能、属性规则、Boss 机制、PVP/PVE 阵容案例，最终用于辅助判断不同场景下的阵容、站位、培养和配置建议。

## 当前阶段

当前先建立知识库结构，后续每次补充资料时按统一格式记录：

- `knowledge/mechanics/`：属性、战斗规则、伤害公式、控制、速度、能量等机制。
- `knowledge/heroes/`：英雄技能、定位、关键属性、常见搭配和克制关系。
- `knowledge/hero enabling/`：英雄赋能、赋能灌注和赋能相关伤害乘区。
- `knowledge/artifacts/`：神器截图、神器等级、简称、属性和效果。
- `knowledge/soulstone/`：水晶截图、词缀组合和适用方向。
- `knowledge/monster/`：魔兽技能、光环、符文强化和队伍增幅。
- `knowledge/bosses/`：Boss 技能、阶段机制、应对重点。
- `knowledge/modes/`：PVP、PVE、活动、Boss 战等玩法场景。
- `knowledge/lineups/`：阵容案例、站位、配置、适用条件和风险。
- `knowledge/templates/`：录入新知识时使用的模板。

## 工作方式

你可以直接把游戏里的资料、截图文字、战斗观察或阵容问题发给我。我会先整理成结构化知识，再在信息足够时给出建议。

阵容建议默认会包含：

- 适用场景
- 推荐阵容和站位
- 核心机制
- 英雄配置重点
- 替代英雄或低配方案
- 风险和待确认点
