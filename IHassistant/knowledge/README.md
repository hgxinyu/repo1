# Knowledge Base

这里存放放置奇兵相关知识。记录时优先保持事实和推论分离：

- `事实`：来自游戏内描述、用户提供截图、已验证战斗结果。
- `推论`：基于事实推导出的机制理解或阵容判断。
- `待确认`：可能正确但还缺验证的信息。

## 目录

- `mechanics/`：通用机制。
- `heroes/`：英雄资料。
- `hero enabling/`：英雄赋能、赋能灌注和相关伤害乘区。
- `artifacts/`：神器资料与截图。
- `soulstone/`：水晶资料与截图。
- `awaken copy/`：觉醒资质词条资料与截图。
- `monster/`：魔兽资料与截图。
- `bosses/`：Boss 资料。
- `starsoul/`：星魂系统资料，待补充。
- `grimoire sigils/`：魔典符印系统资料，待补充。
- `train/`：列车系统资料，待补充。
- `expedition star imprints/`：远征星印系统资料。
- `imprint infusion/`：印痕灌注系统资料与截图。

- `lineups/`：阵容案例。
- `templates/`：录入模板。

## 关键机制文档

- `mechanics/attributes-affixes.md`：属性词缀、基础战斗规则和通用乘区。
- `mechanics/boss-damage-formula.md`：章鱼 Boss 伤害公式和阿斯布主 C 相关乘区。

- `mechanics/control-types.md`：沉默、恐惧、光之封印、双封、变形术、目眩、嘲讽等控制类型和交互。
- `awaken copy/README.md`：觉醒资质等级、词条门槛和各资质数值表。
- `expedition star imprints/README.md`：远征星印数量、套装规则、远见 4 件套和常用 4+2 配置。
- `modes/star-expedition.md`：星域远征场景配置口径。
- `imprint infusion/README.md`：印痕灌注点数规则、利刃/强壁/精神三大类和各分支效果。

## 记录规范

- 新知识尽量注明来源：游戏内文本、战斗观察、用户经验、版本日期。
- 技能描述保留关键数值和触发条件。
- 结论不要混在原始事实里，单独放到“分析”或“建议”部分。
- 对 PVP 和 PVE 分开记录，避免把一个场景的强度结论泛化到另一个场景。
