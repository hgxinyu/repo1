# 星魂系统

这里记录星魂系统的游戏内描述、来源截图和机制整理。星魂资料按星魂类型分目录保存，具体数值以截图日期和截图内等级为准。

## 目录

- `nova/`：新星类星魂。
- `pupil/`：瞳类星魂，待补充。
- `specter/`：幻影类星魂，待补充。
- `bloom/`：花/绽放类星魂，待补充。
- `butterfly/`：蝶类星魂，待补充。
- `netherfish/`：冥鱼类星魂，待补充。
- `plume/`：羽类星魂，待补充。

## 记录口径

- `事实`：仅记录游戏内截图可见文本。
- `待确认`：截图未显示、等级含义、触发顺序、伤害结算乘区等需要实战验证的信息。
- 星魂效果不是英雄技能本体，分析阵容或 Boss 伤害时应明确是否装备了对应星魂。
