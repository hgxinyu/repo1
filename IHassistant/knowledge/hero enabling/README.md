# 英雄赋能

## 基础规则

- 每个英雄可以选择 5 个赋能。
- 每个赋能位只能在 3 个列选项中选择 1 个。
- 用户确认：赋能 1 和赋能 4 使用同一组三选一。
- 表格中每个赋能选项有基础效果、`lv1` 和 `lv2` 子行；当前默认计算时认为 `lv1` 和 `lv2` 都已生效。
- 来源截图：`赋能灌注.jpg`

## 与 Boss 伤害公式相关的赋能

### 绝地

- 来源：赋能 2，第二列。
- 基础效果：
  - 当主动技能或普通攻击对当前生命值高于自身的敌人造成伤害时，额外造成自身造成全伤害的 12%。
- lv1：
  - 额外伤害提高到 18%。
- lv2：
  - 当目标当前生命值高于自身，且超出部分大于等于自身当前生命值 30% 时，对目标额外造成其最大生命值 30% 的伤害，上限为自身攻击力 3000%。
- 用户补充：
  - 原版绝地为 12%，灌注后加成为 18%。
  - 当前默认按赋能灌注后的算法，即 `绝地 18%` 计算。

### 均衡

- 来源：赋能 5，第一列。
- 基础效果：
  - 如果普通攻击或主动技能暴击，恢复自身造成伤害 15% 的生命；否则额外造成 30% 伤害。
- lv1：
  - 暴击时恢复自身生命提高到造成伤害的 22%，否则额外伤害提高到 45%。
- lv2：
  - 如果普通攻击或主动技能未暴击，且目标生命值低于 50%，额外伤害提高 150%。
  - 如果普通攻击或主动技能暴击，额外提高自身 12% 减伤率，持续 2 回合。
- 用户补充：
  - Boss 公式里写的 `均衡 30%` 是未灌注版本。
  - 由于当前默认使用赋能加强版 `lv 灌注`，后续计算按 `均衡 45%` 处理。
  - 当前口径：同一英雄本回合只要有任意一段普通攻击/主动技能伤害没有暴击，`均衡` 就对该回合总伤害触发额外 45% 伤害。
  - 因此多段伤害中，不应只给未暴击的单段加 `均衡`；该回合内其他已暴击段也会一起吃到 `均衡` 的回合总伤害增幅。
