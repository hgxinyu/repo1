# 水晶知识库

每个英雄可以佩戴 1 个水晶。水晶提供固定属性或百分比属性，评估英雄最终属性、速度线、输出和生存时需要计入。

## 词缀方向

### 双爆

- 适用方向：暴击输出。
- 核心价值：同时提供暴击率、暴击伤害和攻击。
- 初步用途：依赖暴击和爆发伤害的输出英雄。
- 风险：如果英雄暴击不足或目标抗暴/暴伤减免高，收益会下降。

### 双血

- 适用方向：生存。
- 核心价值：同时提供固定生命值和百分比生命值。
- 初步用途：前排、承伤、辅助、需要最大生命值支撑技能收益的英雄。
- 风险：不提供速度、攻击或控制相关属性。

### 神技

- 适用方向：技能伤害输出。
- 核心价值：神圣伤害和增加技能伤害率。
- 初步用途：主动技能伤害占比高、且吃技能伤害率的英雄。
- 风险：不提供攻击百分比、速度和生存；需结合英雄伤害类型确认是否吃技能伤害率。

### 攻血

- 适用方向：均衡。
- 核心价值：攻击和生命值兼顾。
- 初步用途：既需要输出又需要站场的英雄。
- 风险：没有速度和特殊输出词缀，爆发上限不如专门输出水晶。

### 双攻

- 适用方向：攻击面板。
- 核心价值：固定攻击和百分比攻击叠加。
- 初步用途：技能、治疗、护盾或其他效果强依赖攻击面板的英雄。
- 风险：不提供速度、生存、暴击或神圣伤害。

### 神攻

- 适用方向：攻击和神圣伤害。
- 核心价值：神圣伤害、固定攻击和百分比攻击兼顾。
- 初步用途：攻击面板重要，同时需要神圣伤害补充的输出英雄。
- 风险：不提供速度和生命；是否优于双攻/神技要看英雄伤害构成。

### 速攻

- 适用方向：速度线和输出。
- 核心价值：+130 速度并提供攻击。
- 初步用途：需要抢先手、卡速度线或先手释放技能的输出/功能英雄。
- 风险：不提供生存，速度收益取决于具体对局速度线。

### 速血

- 适用方向：速度线和生存。
- 核心价值：+130 速度并提供生命值。
- 初步用途：需要先手行动但更重视存活的辅助、控制或功能英雄。
- 风险：不提供攻击输出。

## 记录要求

- 后续新增水晶截图时，优先记录截图文件名、简称、所有词缀和适用方向。
- 阵容建议中如果涉及速度顺序，必须把水晶速度词缀计入最终速度。
- 输出建议中要区分攻击、暴击、神圣伤害、技能伤害率，不要默认所有输出英雄都适合同一个水晶。
