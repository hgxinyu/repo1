# 觉醒资质词条

## 来源

- 来源截图：`Weixin Image_2026-05-27_145821_020.jpg`
- 用户说明：
  - 每个英雄都可以觉醒资质。
  - 不同档次的资质可以携带不同属性词条。
  - 表格行是词条，列是资质等级。
  - `免控率` 行以下是所有资质都可以有的词条。
  - `挑战者` 到 `阵营铁律` 是 `A-` 以上才可以有的词条。
  - `猎手专注` 到 `免疫眩晕` 是 `S` 资质以上才可以有的词条。
  - 更上方是 `X` 专属词条。
  - 表格里的黑色 block 表示该资质不适用该词条，可以忽略，不按缺失数值处理。

## 词条门槛

### X 专属词条

| 词条 | 效果 |
| --- | --- |
| 庇佑身躯 | 自身被施加属性增益效果时，使该增益最终效果值提高 33%。 |
| 背水之势 | 当自身生命低于 30% 时，有 25% 概率免疫受到的直接伤害。 |
| 不屈战魂 | 自身生命值百分比越低，自身造成全伤害和全伤害减免越高，最高获得 25% 造成全伤害增加和 25% 全伤害减免。 |
| 洁净心流 | 自身被施加控制效果时，使该效果持续回合数 -1。 |

### S 资质以上词条

| 词条 | 效果 |
| --- | --- |
| 免疫燃烧/流血/中毒 | 免疫燃烧、流血、中毒。 |
| 免疫控制 | 免疫眩晕、冰冻、沉默、石化、缠绕、恐惧、嘲讽、目眩、变形术、光之封印。 |
| 碎舰者 | 对带有护盾的敌人造成的伤害增加 40%。 |
| 猎手专注 | 若自身暴击大于或等于 50%，则使自身造成的全伤害增加 30%。 |

### A- 资质以上词条

| 词条 | 效果 |
| --- | --- |
| 阵营铁律 | 阵营克制增益对所有类型的伤害生效，治愈诅咒除外。 |
| 神圣招架 | 基于格挡的 100% 获得神圣格挡。 |
| 神圣格挡 | 使格挡成功后的减伤对所有类型的伤害生效。 |
| 神圣铠甲 | 基于护甲的 100% 获得神圣护甲。 |
| 神圣护甲 | 使护甲无法减免的伤害也能被护甲影响，治愈诅咒除外。 |
| 巨人杀手 | 对目标造成伤害时，目标最大生命值每高于自身 10%，造成的伤害提高 2%，最高不超过 100%。 |
| 挑战者 | 造成伤害时，若目标生命值高于或等于 70%，造成的伤害提高 30%。 |

## 机制分析口径

- 评估英雄最终属性时，觉醒资质词条应和装备、神器、水晶、赋能、魔兽等一起计入。
- 输出英雄优先关注：
  - 增加技能伤害率。
  - 神圣伤害。
  - 精准。
  - 暴击率、暴击伤害。
  - 破甲。
  - X / S / A 段里的特殊增伤词条，例如 `猎手专注`、`碎舰者`、`巨人杀手`、`挑战者`。
- 控制英雄优先关注：
  - 控制命中率。
  - 抵消免控率。
  - 免控率。
  - `洁净心流` 这类减少控制持续回合的高资质词条。
- 生存英雄优先关注：
  - 减伤率。
  - 暴击伤害减免。
  - 持续伤害减免。
  - 格挡、护甲。
  - `不屈战魂`、`背水之势`、`神圣格挡`、`神圣护甲` 等特殊防御词条。
- 真实伤害和惩戒伤害分析时，要特别留意 `阵营铁律`、`神圣格挡`、`神圣护甲` 这类改变属性适用范围的词条。

## 待确认

- `阵营铁律` 对惩戒伤害是否也生效；当前只按“所有类型的伤害，治愈诅咒除外”原文记录。
