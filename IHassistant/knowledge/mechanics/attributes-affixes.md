## 正常战斗基础结构

- 每队最多出战 6 个英雄。
- 阵容站位按 1 到 6 号位记录。
- 每个队伍可以携带 1 个魔兽；魔兽技能、光环和符文强化会影响全队输出、生存和速度等属性。
- 每个英雄可以穿装备和佩戴神器。
- 神器会给英雄附加属性；计算英雄最终速度、攻击、血量等属性时，需要计入神器提供的属性。
- 每个英雄可以佩戴 1 个水晶；计算英雄最终速度、攻击、血量、暴击、神圣伤害等属性时，需要计入水晶词缀。

## 攻击

- 作用：提升英雄造成的伤害。
- 计算位置：攻击总值，先加后乘。
- 普攻伤害公式：
  - `普攻伤害 = 基础伤害 x (1 - 减伤) x (1 - 格挡减伤) x (1 - 科技减伤) x (1 - 印记减伤)`

## 血量

- 作用：提升英雄的基础血量。
- 计算位置：血量总值，先加后乘。

## 护甲

- 作用：提升英雄的防御力。
- 图片公式：
  - `护甲减伤率 = 护甲 / (200 + 20 x (等级 - 1))`
- 待确认：
  - 是否存在护甲减伤上限。
  - 当前版本公式是否仍然适用。

## 速度

- 作用：决定英雄出手顺序。
- 规则：
  - 英雄出手顺序按照最终速度决定。
  - 速度高的英雄先出手。
  - 双方速度相同时，攻方先出手。
  - 同方速度相同时，位置靠前的英雄先出手。

## 增加技能伤害率

- 作用：直接加到主动技能伤害的百分比中。
- 限制：
  - 通常仅对传统伤害有效。
  - 对真实伤害和惩戒伤害默认不生效，除非技能文本或实测另有说明。
- 图片公式：
  - `技能伤害 = 普攻伤害 x 技能伤害率`
- 持续伤害：
  - `技能伤害率 / (持续伤害时效 + 1)`
- 待确认：
  - “持续伤害时效 + 1”的具体含义和适用范围。

## 精准

- 作用：
  - 降低目标格挡几率。
  - 每 1% 精准增加 0.3% 伤害。
  - 最高增加 45% 伤害。
- 机制补充：
  - 精准与格挡先做加减法抵消。
  - 精准除了抵消格挡外，还会转换为伤害加成。
- 限制：
  - 精准增伤通常仅对传统伤害有效。
  - 真实伤害和惩戒伤害默认不吃精准增伤。
- 图片备注：
  - `精准 150%` 达到增伤上限。

## 格挡

- 作用：
  - 决定格挡几率。
  - 格挡成功减少 30% 伤害。
  - 如果是暴击伤害，额外减少 20%。
- 限制：
  - 格挡减伤通常仅对传统伤害有效。
  - 真实伤害和惩戒伤害默认不判定格挡。
- 图片公式：
  - `有效格挡 = D 格挡 - A 精准`
  - 负值取零。

## 暴击

- 作用：基础暴击造成 150% 伤害。
- 图片公式：
  - `暴击伤害 = 伤害 x (1.5 + 2 x 暴击伤害率)`
- 图片备注：
  - `暴击伤害率` 上限：150%。
- 待确认：
  - 公式中的 `2 x 暴击伤害率` 是否为当前版本实际规则，需结合游戏内战斗验证。

## 暴击伤害值

- 作用：增加暴击伤害。
- 面板限制：
  - 实际面板最高暴击伤害值为 150%。

## 破甲

- 作用：百分比抵消目标护甲值。
- 上限：100%。
- 图片备注：
  - `100% 破甲` 即该攻击者攻击防守者时，防守者护甲减伤率为零。
- 机制补充：
  - 破甲与护甲不是普通加减法抵消，而是对目标护甲减伤做百分比消解。
  - 当前 Boss 公式中表现为 `护甲减伤 * (1 - 护甲减免) * (1 - 破甲)`。
- 用户确认：
  - `破甲` 和 `护甲减免/降护甲` 暂按同一类护甲抵消效果理解。
  - `破甲 100%` 可以理解为 `降护甲 100%`。

## 免控率

- 作用：免除控制的几率。
- 控制类型：
  - 当前已整理 10 种控场类型：沉默、恐惧、光之封印、冰冻、石化、眩晕、缠绕、变形术、目眩、嘲讽。
  - 具体分类、封锁范围和死亡技能交互见 `control-types.md`。
- 规则：
  - 免控率和控制率分别计算。
  - 例如 50% 控制率对 0% 免控目标必然有可能控制。
- 图片公式：
  - `被控率 = A 的控制率 x (1 - D 的免控率) x (1 - D 的科技免控率)`
- `抵消目标免控率`：
  - 作用：攻击者侧属性，用于抵消目标的普通免控率。
  - 计算时先降低目标普通免控率，再代入被控率公式。
  - `有效免控率 = max(0, D 的免控率 - A 的抵消目标免控率)`
  - `被控率 = A 的控制率 x (1 - 有效免控率) x (1 - D 的科技免控率)`
  - 例如攻击者控制率 100%，抵消目标免控率 20%，目标普通免控率 50%，目标科技免控率 0% 时，被控率为 `100% x (1 - 30%) = 70%`。
- 待确认：
  - 控制率是否存在不同控制类型的额外判定。

## 减伤率

- 作用：降低受到的伤害。
- 上限：75%。
- 限制：
  - 减伤仅对基于攻击者自身攻击力计算的伤害有效。
- `抵消减伤率`：
  - 作用：攻击者侧属性，用于抵消目标的普通减伤率。
  - 计算时先降低目标普通减伤率，再代入减伤乘区。
  - `有效减伤率 = max(0, D 的减伤率 - A 的抵消减伤率)`
  - 对应乘区可记为 `1 - 有效减伤率`。
- 注意：
  - 减伤率和全伤害减免不是同一个属性。
  - `抵消减伤率` 不等于 `抵消全伤害减免`。

## 造成全伤害 / 全增伤

- 作用：提高自身最终造成的全伤害。
- 常见描述：
  - `造成全伤害增加`
  - `全伤`
  - `全增伤`
- 计算关系：
  - 攻击者的全增伤与防守者的全伤害减免不是直接减法抵消。
  - 二者按独立乘区计算：

```text
伤害 = 原始输出 * (1 + A 的全增伤) * (1 - D 的有效全伤害减免)
```

- 例子：
  - 攻击者全增伤 10%，防守者有效全伤害减免 6%。
  - 最终比例是 `1 * (1 + 10%) * (1 - 6%) = 103.4%`。
  - 不是 `1 + 10% - 6% = 104%`。
- 机制结论：
  - 全增伤不会“抵消”全伤害减免。
  - 若要抵消全伤害减免，需要看是否有单独的 `抵消全伤害减免` 属性。
- 用户补充：
  - 巨人杀手、挑战者、猎手专注、碎舰者、神能压制增伤和回合增伤可按“全增伤类属性”理解。
  - 这些属性与普通全增伤的增伤模式相似，但互相之间按叠乘处理，不是先相加再统一乘一次。
  - 根据伤害量再次造成的真实伤害，可能会再次吃到这些全增伤类乘区。

## 全伤害减免 / 全减伤

- 作用：降低自身受到的全类型伤害。
- 常见描述：
  - `全伤害减免`
  - `全减伤`
- 计算关系：
  - 与攻击者全增伤按乘法关系结算，见 `造成全伤害 / 全增伤`。
- `抵消全伤害减免`：
  - 作用：攻击者侧属性，用于抵消目标的全伤害减免。
  - 计算时先降低目标全伤害减免，再代入全减伤乘区。
  - `有效全伤害减免 = max(0, D 的全伤害减免 - A 的抵消全伤害减免)`
  - 对应乘区可记为 `1 - 有效全伤害减免`。
- 注意：
  - `造成全伤害增加`、`造成全伤害降低`、`全伤害减免`、`抵消全伤害减免` 是不同概念。
  - 文章实测结论确认：攻方全增伤与守方全减伤不做直接相减。
- 来源归类：
  - 神器可以提供全伤害减免、按攻击力减伤或伤害限制类效果。
  - 列车系统可能提供减伤或全伤害减免来源；当前缺少具体数值，先作为防守端减伤来源预留。
  - 神能压制也可能提供最终伤害减免。
  - 粉石墨 / 石墨类限制按用户补充会作用于每段伤害，且计算顺序为先石墨限制，再回合增伤，最后攻击减伤。

## 资质增伤

这里指觉醒资质词条提供的特殊增伤，不是资质等级本身自动增伤。

已整理的资质增伤词条包括：

- `巨人杀手`：目标最大生命值每高于自身 10%，造成伤害提高 2%，最高 100%。
- `挑战者`：目标生命值不低于 70% 时，造成伤害提高 30%。
- `猎手专注`：自身暴击大于或等于 50% 时，造成全伤害增加 30%。
- `碎舰者`：对带护盾敌人造成伤害增加 40%。
- `不屈战魂`：自身生命越低，造成全伤害和全伤害减免越高，最高各 25%。

计算口径：

- 这些词条属于高资质可携带词条，具体门槛见 `knowledge/awaken copy/README.md`。
- 用户补充将巨人杀手、挑战者、猎手专注、碎舰者归入“全增伤类属性”。
- 当前按独立乘区处理；是否与完全同名的其他来源叠加或互斥，后续按游戏文本和实测修正。

## 神能压制增伤 / 减伤

- 神能压制是最终类乘区，会作用在伤害计算后段。
- 用户补充称每一段伤害都会触发神能压制；多段技能会逐段增伤或减伤。
- 根据伤害量触发的追加伤害也可能再次吃神能压制，因此会导致均衡、绝地、暗战 Boss 额外伤害等机制出现多次放大。
- 旧 Boss 公式中的 `1 + (英雄神能差值 + 集体神能差值) / 10` 先保留为截图公式口径；`神能压制` 与该差值公式的精确关系仍待确认。

## 攻击减伤与伤害限制

- `攻击减伤` 指按防守方攻击力或某个攻击值减少受到伤害的效果。
- 用户补充：攻击减伤会作用于每段伤害，不管伤害类型。
- `石墨` / `粉石墨` 类效果属于伤害限制或伤害上限类效果，用户补充称它也会作用于每段伤害，包括真实伤害。
- 当前顺序口径：

```text
先计算石墨限制，再计算回合增伤，最后计算攻击减伤
```

- 神器减伤和列车减伤目前先按防守端减伤来源归类；如果后续有具体截图或文本，需要拆成全伤害减免、攻击减伤、伤害限制或其他独立乘区。

## 抵消类属性

- 目前已记录或提到的抵消类属性：
  - 抵消免控率。
  - 抵消闪避。
  - 抵消减伤率。
  - 抵消全伤害减免。
- 通用口径：
  - 抵消类属性通常与对应防御属性做加减法关系。
  - 计算时先得到对应属性的有效值，再进入该属性自己的乘区或判定公式。
- 已知例外/区别：
  - 精准与格挡做抵消，但精准还额外转换为伤害加成。
  - 破甲与护甲不是普通加减法抵消，而是按百分比消解护甲减伤。
- 待确认：
  - `抵消闪避` 的当前版本公式和适用范围。

## 神圣伤害

- 作用：额外造成无视护甲的伤害。
- 但会被减伤率降低。
- 限制：
  - 通常只对传统伤害有效。
  - 真实伤害和惩戒伤害默认不吃神圣伤害。
- 图片公式：
  - `基础伤害 = A 的攻击力 x (1 - D 的护甲减伤率 x (1 - A 的破甲率)) + A 的攻击力 x A 的神圣伤害率 x 0.7`

## 对指定职业伤害加成

- 作用：提高己方英雄对指定职业造成的伤害。
- 限制：
  - 通常仅对传统伤害有效。

## 持续伤害减免

- 作用：减少受到的所有持续伤害。
- 机制补充：
  - 文章将持续伤害减免归为类似百分比减免/消解类关系，而不是普通抵消词条。

## 暴击伤害减免 / 受到暴击伤害增加

- 作用：
  - 暴击伤害减免：降低受到的暴击伤害。
  - 受到暴击伤害增加：提高目标受到的暴击伤害。
- 机制补充：
  - 文章提到二者按加减法关系处理。
- 待确认：
  - 当前版本的完整暴击伤害减免公式。
  - 该关系与女王 `深渊侵蚀` 的“受到暴击伤害提高”是否完全同一乘区。

## 能量

### 英雄能量

- 初始能量：50 点。
- 普通攻击增加：50 点。
- 被击增加：10 点。
- 被暴击增加：20 点。
- 轮到英雄出手时：
  - 能量不满 100，使用普通攻击。
  - 能量满 100，使用大招/主动技能。
- 超过 100 点时：
  - 每溢出 1 点，按 1% 技能伤害率对主动技能进行加成。

### 魔兽能量

- 每队携带的魔兽有独立能量。
- 起始能量：0 点。
- 每次任意英雄释放大招/主动技能时，魔兽能量 +10。
- 每过 1 回合，魔兽能量 +20。
- 魔兽能量满 100 时，释放魔兽技能。

## 受到治疗效果提升

- 作用：提高自身受到的治疗效果。

## 造成治疗效果提升

- 作用：提高自身造成的治疗效果。
