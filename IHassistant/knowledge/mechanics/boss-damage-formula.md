## 场景限制

- 基础公式用于章鱼 Boss / 星域远征 Boss 的传统伤害估算。
- 远征星印、远见套、卡尔/命运/布尔等星印套装加成只适用于星域远征/远征 Boss 场景。
- 如果计算普通 Boss、非远征 PVE 或 PVP，不要默认计入星印相关加成。

## 截图公式整理

对章鱼产生的总输出：

```text
总输出 =
{ 攻击 * [1 - 护甲减伤 * (1 - 护甲减免) * (1 - 破甲)]
  + 攻击 * 神圣伤害 * 70%
}
* (主动技能系数 + 技能伤害率 + 溢出 100 能量后对等计算为技能伤害率)
* (1 + 精准 * 0.3%)
* (1 + 观心印记)
* (暴击 1.5 倍 + 爆伤 * 2)
* (1 + 女皇侵蚀)
* (1 + 魔兽增伤 * 层数)
* (1 + 对持续伤害增伤 * 回合)
* (1 + 全伤)
* (1 + 极鹿角 * 回合)
* [1 + 均衡 45% * (1 + 全伤) * (1 + 极鹿角 * 回合)]
* [1 + 绝地 18% * (1 + 全伤) * (1 + 极鹿角 * 回合)]
* 技能段数
* [1 + (英雄神能差值 + 集体神能差值) / 10]
```

## 用户补充的扩展公式口径

2026-06-03 用户转述的伤害算法文章把伤害拆为 `非真实伤害`、`真实伤害` 和 `根据伤害量再次造成的真实伤害`。这部分先作为扩展口径记录，后续遇到具体英雄和实战再校正。

### 非真实伤害

非真实伤害可理解为传统攻击伤害。简化拆分：

```text
非真实伤害 =
[
  攻击力 * (技能系数 + 技能伤害率 + 主动技能溢出能量)
  * 护甲相关乘区
  +
  攻击力 * (技能系数 + 技能伤害率 + 主动技能溢出能量)
  * 神圣伤害 * 0.7
  * 神圣铠甲相关护甲乘区
]
* 普通减伤/阵营克制乘区
* 格挡减伤乘区
* 职业增伤/职业减伤乘区
* 印记增伤/印记减伤乘区
* 持续伤害减免乘区
* 全伤与全减伤乘区
* 精准增伤乘区
* 暴击与暴击伤害乘区
* 状态增伤乘区
* 全增伤类属性乘区
* 回合增伤
- 攻击减伤
```

其中需要补进旧公式的来源项：

- 资质增伤：巨人杀手、挑战者、猎手专注、碎舰者、不屈战魂等觉醒词条。
- 神能压制增伤 / 减伤：属于最终类乘区，每段伤害都会结算。
- 神器减伤：可能表现为全伤害减免、攻击减伤、护盾、伤害限制或石墨类限制，必须按具体神器文本拆分。
- 列车减伤：当前缺少具体数值，先作为防守端全减伤/减伤来源预留。
- 石墨限制：用户补充称会作用于每段伤害，包括真实伤害。
- 攻击减伤：会作用于每段伤害，不管伤害类型。

### 真实伤害

用户补充的真实伤害口径：

```text
真实伤害 =
技能描述中的伤害基准
* 系数
* 全增伤类属性
* (1 - 持续伤害减免，如果适用)
* 神圣铠甲相关护甲乘区
* 神能压制乘区
* 回合增伤
- 攻击减伤
```

注意：

- 真伤通常不吃神圣伤害、精准、暴击、格挡、观心印记等传统伤害乘区。
- 如果目标有 `神圣铠甲 / 神圣护甲`，护甲减伤可作用到真实伤害和神圣伤害部分，但会被破甲无视；该规则来自用户补充，仍需实测确认。
- 石墨限制和攻击减伤按用户补充会作用于每段伤害。

### 根据伤害量再次造成的真实伤害

部分技能不是按攻击力或普通系数算，而是按“已造成伤害量”再次造成伤害。用户补充把这类也归入真实伤害，例如：

- 暗战对 Boss 的额外伤害。
- `均衡`。
- `绝地`。

当前记录口径：

```text
伤害量真伤 =
[
  基准伤害量
  * 全增伤类属性
  * (1 - 持续伤害减免，如果适用)
  * 神圣铠甲相关护甲乘区
  * 神能压制乘区
  * 回合增伤
  - 攻击减伤
]
```

因为基准伤害量本身可能已经吃过全增伤类属性，所以这类追加伤害会形成多次放大。

### 石墨、回合增伤、攻击减伤顺序

用户补充的顺序：

```text
先计算石墨限制
再计算回合增伤
最后计算攻击减伤
```

待确认：

- 石墨限制的正式名称、具体神器档位和上限。
- 攻击减伤的来源和是否存在下限。
- 回合增伤是否统一为超过 15 回合后 `1.15^(回合数 - 15)`，以及该增伤是否在所有 Boss/PVE 场景一致。

## 与已学机制的对照

### 适用伤害类型

这张 Boss 公式主要适用于 `传统伤害`，也就是基于攻击者自身攻击力、会吃神圣伤害/破甲/精准/暴击/减伤/观心印记等乘区的伤害。

- `真实伤害` 不能直接套完整公式：
  - 通常不吃神圣伤害、破甲、精准、护甲、减伤率、暴击、格挡、观心印记、魔兽增伤等。
  - 通常仍需考虑全伤、全减伤、神能差值、护盾和闪避。
- `惩戒伤害` 更不能套完整公式：
  - 常规攻防属性、全伤、全减伤、护盾等通常都不改变惩戒伤害。
  - 当前按只受神能差值影响记录。

详见 `damage-types.md`。

### 基础伤害

截图中的基础部分：

```text
攻击 * [1 - 护甲减伤 * (1 - 护甲减免) * (1 - 破甲)]
+ 攻击 * 神圣伤害 * 70%
```

和已有 `神圣伤害` 记录基本一致：

```text
基础伤害 =
A 的攻击力 * (1 - D 的护甲减伤率 * (1 - A 的破甲率))
+ A 的攻击力 * A 的神圣伤害率 * 0.7
```

新增差异：

- 新公式多了 `(1 - 护甲减免)` 这一项。
- 用户确认：`护甲减免` 和 `破甲` 暂按同一类护甲抵消/降护甲效果理解；破甲 100% 可以理解为降护甲 100%。
- 记录口径：
  - 后续如果技能描述写“破甲 100%”或“降低护甲 100%”，默认都按抵消目标护甲减伤处理。
  - 若后续发现两个词在游戏内有不同乘区或叠加方式，再单独修正。

### 技能伤害率与能量溢出

截图把以下三项放在同一个加法乘区：

```text
主动技能系数 + 技能伤害率 + 溢出 100 能量后对等计算为技能伤害率
```

这和已有机制能对上：

- 主动技能有自己的倍率，例如阿斯布 `灵魂战栗` 是 `1200%`。
- `增加技能伤害率` 直接加到主动技能伤害百分比中。
- 能量超过 100 时，每溢出 1 点，按 1% 技能伤害率对主动技能加成。

对阿斯布举例：

- 如果一段主动是 `1200%`，阿斯布有 `300% 增加技能伤害率`，且释放时能量为 120，则这一乘区可理解为：
  - `1200% + 300% + 20%`
- 如果技能描述为 `2 次 1200%`，则公式里后面再乘 `技能段数 = 2`。

### 精准

截图：

```text
1 + 精准 * 0.3%
```

和已有机制一致：

- 每 1% 精准增加 0.3% 伤害。
- 旧图记录最高增加 45% 伤害，即精准 150% 到增伤上限。

### 观心印记

公式中的 `观心印记` 来源于观心者（Heart Watcher / HW）的 `观心者印记`。

- 主动技能 `心灵拷问` 施加的印记：
  - 目标被攻击时额外受到 45% 伤害。
- 普通攻击被动 `弱点打击` 施加的印记：
  - 目标被攻击时额外受到 35% 伤害。
- 印记可叠加。
- 主动 45% 和普攻 35% 直接相加，共用 300% 上限。
- 对单一目标最高叠加至 300% 额外受伤。
- 当前默认：`观心者印记` 不会被清除。

当前公式口径：

```text
观心印记 = 目标身上观心者印记累计增伤，最高 300%
```

详见 `knowledge/heroes/HW/heart-watcher.md`。

### 女皇侵蚀

公式中的 `女皇侵蚀` 来源于血腥女王-哈尔拉（Halora / SQH / 女王）的 `深渊侵蚀`。

- 女王主动技能 `深渊降临` 对所有敌人施加 1 层 `深渊侵蚀`。
- `深渊侵蚀` 效果：
  - 被侵蚀英雄暴击伤害降低 24%。
  - 被侵蚀英雄受到的暴击伤害提高 24%。
- 叠加规则：
  - 可以叠加。
  - 每层 24% 直接相加。
  - 上限为增加 200% 伤害。
  - Boss 战斗通常会打满 15 回合，当前默认按最终叠满 `女皇侵蚀 = 200%` 计算。
- 升华强化：
  - `渊之升华`：被侵蚀英雄暴击降低 10%。
  - `崇之升华`：施加 `深渊侵蚀` 时，有 25% 概率额外施加 1 层。

当前公式口径：

```text
女皇侵蚀 = 目标身上深渊侵蚀累计增伤，最高 200%；Boss 战默认按 200%
```

详见 `knowledge/heroes/queen/queen.md`。

### 暴击与暴击伤害

截图：

```text
暴击 1.5 倍 + 爆伤 * 2
```

和已有暴击公式一致：

```text
暴击伤害 = 伤害 * (1.5 + 2 * 暴击伤害率)
```

需要注意：

- 该乘区只在实际暴击时成立。
- 如果没有暴击，理论上应不套用 `1.5 + 爆伤 * 2`，或者按非暴击乘区处理；截图没有写非暴击分支。

#### 暴击率调控口径

如果不考虑 `均衡`，暴击率越高期望伤害越高，因为暴击乘区为：

```text
暴击乘区 = 1.5 + 2 * 暴击伤害率
```

如果带 `均衡`，则暴击和未暴击有分支：

- 全部暴击：吃暴击乘区；`均衡` 走回血分支，不提供额外伤害。
- 至少一段未暴击：未暴击段不吃暴击乘区，同时触发 `均衡` 额外伤害。
- 用户补充：同一英雄本回合只要有任意一段普通攻击/主动技能伤害没有暴击，`均衡` 就对该回合总伤害增加 45%（灌注后）。它不是只加未暴击段，也不是只加某一段伤害。

15 回合、鹿角按极 1 鹿角 `9% * 回合` 计算时：

```text
均衡增伤乘区 B = 1 + 45% * (1 + 全伤) * (1 + 9% * 15)
              = 1 + 1.0575 * (1 + 全伤)
```

暴击是否优于未暴击，取决于：

```text
1.5 + 2 * 暴击伤害率 > 1 + 1.0575 * (1 + 全伤)
暴击伤害率 > 27.875% + 52.875% * 全伤
```

因此单段伤害的默认判断：

- 不带 `均衡`：暴击率尽量高。
- 带 `均衡` 且暴击乘区高于未暴击均衡乘区：暴击率尽量高。
- 带 `均衡` 且未暴击均衡乘区高于暴击乘区：暴击率不宜继续堆高。

常用阈值参考：

| 全伤 | 暴击优于未暴击均衡所需暴击伤害率 |
| ---: | ---: |
| 0% | 27.875% |
| 50% | 54.3125% |
| 100% | 80.75% |
| 150% | 107.1875% |
| 200% | 133.625% |
| 231% 以上 | 超过 150% 暴击伤害上限，未暴击均衡更优 |

单段伤害时，如果不带 `猎手专注`，没有必须卡 `50%` 的硬门槛；理论最优可以低于 50%。如果带 `猎手专注`，且判断未暴击均衡更优，暴击率才建议控制在 `50%` 刚过线的区域，通常记为 `50%~55%`。

多段伤害时，因为任意一段未暴击就能让本回合总伤害吃到 `均衡`，暴击率可以比单段更高。

假设：

- 暴击伤害率满值 `150%`，暴击乘区 `M = 1.5 + 2 * 150% = 4.5`。
- 同一英雄本回合计入 `n` 段伤害，每段独立按面板暴击率 `c` 判定。
- 只要任意一段未暴击，本回合总伤害均吃 `均衡增伤乘区 B`。

则每段平均期望乘区可按：

```text
E = c^n * M + c * (1 - c^(n-1)) * M * B + (1 - c) * B
```

最优暴击率近似为：

```text
c* = [B * (M - 1) / (n * M * (B - 1))] ^ [1 / (n - 1)]
```

其中 `M = 4.5`。常用口径：

- `n = 2` 时，`c* = 3.5B / [9 * (B - 1)]`。
- 当 `全伤 = 0%`，`B = 2.0575`，两段最优暴击率约 `75.7%`。
- 当 `全伤 = 100%`，`B = 3.115`，两段最优暴击率约 `57.3%`。
- 当 `全伤 = 200%`，`B = 4.1725`，两段最优暴击率约 `51.1%`。
- 当 `全伤 >= 231%`，两段理论最优会低于 `50%`；不带 `猎手专注` 时可以低于 50%，带 `猎手专注` 时才需要压回 `50%~55%`。
- `n >= 3` 时，应按段数重算，不应简单沿用两段结论。

满爆伤、不带 `猎手专注` 的常用最优暴击率参考：

| 全伤 | 2 段 | 3 段 | 4 段 | 5 段 | 6 段 | 8 段 |
| ---: | ---: | ---: | ---: | ---: | ---: | ---: |
| 0% | 75.7% | 71.0% | 72.3% | 74.2% | 75.9% | 78.8% |
| 50% | 63.4% | 65.0% | 68.2% | 71.0% | 73.3% | 76.9% |
| 100% | 57.3% | 61.8% | 65.9% | 69.2% | 71.8% | 75.8% |
| 150% | 53.6% | 59.8% | 64.5% | 68.0% | 70.9% | 75.0% |
| 200% | 51.1% | 58.4% | 63.5% | 67.3% | 70.2% | 74.5% |
| 231% | 50.0% | 57.7% | 63.0% | 66.9% | 69.9% | 74.3% |
| 300% | 48.1% | 56.6% | 62.2% | 66.2% | 69.3% | 73.9% |

#### 阿斯布初始暴击率倒推

计算阿斯布初始暴击率时，需要把战斗内暴击率来源倒扣出去。

当前已记录的常见来源：

- 凤凰符文强化：全队 `+15% 暴击率`，按常驻处理。
- 凤凰 Lv.330 灵契强化：受到凤凰治疗后，对燃烧目标释放普通攻击或主动技能时，那次伤害暴击增加 30%，持续 3 回合；当前按条件性 `对燃烧目标暴击率 +30%` 记录。
- 阿斯布主动净之升华：`灵魂战栗 +10% 暴击`，只对主动技能判定生效。
- 阿斯布 `致命抓握`：普通攻击后自身 `+24% 暴击率`，持续 3 回合；渊之升华后持续 4 回合。因为持续多回合，计算 Boss 输出时需要考虑可能叠加；如果再次满足触发条件，且前一层加成仍有效，则继续叠加。每层都有自己的持续回合限制，不是永久叠加，也不是只刷新持续时间。叠加 2 层时覆盖值为 `24% * 2 = 48%`。
- 阿斯布 `死亡扩散` 渊之升华：敌方死亡时额外 `+10% 暴击率`，Boss 单体场景不稳定，默认不计入。
- 远征星印 `卡尔 2 件套`：`+10% 暴击率`，只在星域远征/远征 Boss 场景计入；远征常用星印组合里通常会触发。
- 远征星印 `远见 4 件套`：只在星域远征/远征 Boss 场景计入；主动技能后自身 `+30% 暴击率`、`+100% 暴击伤害`，持续 2 回合；用户确认从本次主动结束后开始计算，不作用于本次主动技能的后续段数。
- 赋能暴击率：用户 2026-06-03 补充，可选择带或不带 `+35% 暴击率`；当前暂按入场/静态暴击来源处理，如果点了 `暴击 -50% / 攻击 +50%`，先假定它会被进场 `-50%` 一起处理，待确认。

倒推公式：

```text
初始暴击率 = 目标实际暴击率 - 常驻星印暴击率 - 凤凰 15% - 主动净升华 10% - 致命抓握覆盖值 - 其他覆盖中暴击率
```

其中：

- 常用远征星印组合触发 `卡尔 2 件套` 时，`常驻星印暴击率 = 10%`。

### 全伤

截图把 `全伤` 作为单独乘区：

```text
1 + 全伤
```

这说明“造成全伤害增加”类效果与技能系数、精准、暴击等乘区相乘。

新文章补充：

- `全伤` / `全增伤` 与目标的 `全伤害减免` / `全减伤` 不是减法抵消。
- 二者按独立乘区计算：

```text
伤害 = 原始输出 * (1 + 攻击者全伤) * (1 - 目标有效全伤害减免)
```

- 例如攻方全伤 10%、守方有效全伤害减免 6%，最终比例是 `1.10 * 0.94 = 1.034`，不是 `1 + 10% - 6%`。
- 如果攻击者有 `抵消全伤害减免`，则先计算：

```text
目标有效全伤害减免 = max(0, 目标全伤害减免 - 攻击者抵消全伤害减免)
```

- 当前章鱼 Boss 公式截图没有单独列出 Boss 的全伤害减免乘区；后续遇到带全减伤的 Boss 或 PVP 目标时，需要把该乘区补进去。

- 资质增伤：巨人杀手、挑战者、猎手专注、碎舰者、不屈战魂等。
- 神能压制增伤 / 减伤。
- 回合增伤。
- 神圣铠甲扩展后的护甲减伤影响。

当前公式口径：

- 这些属性和普通全伤的增伤模式相似，但彼此按叠乘处理。
- 不要把它们写成一个加法总和。
- 对根据伤害量再次造成的真伤，这些属性可能被重复计算。

### 极鹿角

截图单独写了：

```text
1 + 极鹿角 * 回合
```

与已整理鹿角效果能对上：

- 极 1 鹿角 `鹿神之息 - 闪烁` 的 `狂野之力`：每回合结束时，使自身造成的全部伤害提高 9%。
- 极境鹿角 `鹿神之息 - 璀璨` 的 `狂野之力`：每回合结束时也提高自身造成的全部伤害 9%。
- 用户确认：
  - 截图公式中的“粉鹿角”是 typo，应理解为“极鹿角”。
  - 后续提到“鹿角”时，统一按极 1 鹿角的增伤系数计算。

### 均衡与绝地

截图中有两个额外乘区：

```text
[1 + 均衡 45% * (1 + 全伤) * (1 + 极鹿角 * 回合)]
[1 + 绝地 18% * (1 + 全伤) * (1 + 极鹿角 * 回合)]
```

这两个乘区比较特殊：

- 它们不是简单的 `1 + 45%` 或 `1 + 18%`。
- 它们内部还再次乘了 `(1 + 全伤)` 和 `(1 + 极鹿角 * 回合)`。
- 这意味着 `均衡`、`绝地` 可能是额外伤害/追加伤害类机制，而不是普通增伤。

- 用户确认：
  - `绝地` 来自赋能 2 第二列。
  - `均衡` 来自赋能 5 第一列。
  - 原公式里的 `绝地 12%` 是未灌注版本；当前默认使用赋能灌注后的算法，按 `18%` 计算。
  - 原公式里的 `均衡 30%` 是未灌注版本；当前默认使用赋能加强版 `lv 灌注`，按 `45%` 计算。
  - 每个赋能选项的 `lv1` 和 `lv2` 子行默认都按已生效处理。
  - `均衡` 的触发按回合总伤害口径处理：同一英雄本回合只要有任意一段普通攻击/主动技能伤害未暴击，该回合总伤害都吃灌注后 45% 额外伤害。

- 用户 2026-06-03 进一步补充：
  - `均衡` 和 `绝地` 都是根据伤害量再次造成伤害，当前按伤害量真伤口径记录。
  - 先触发 `均衡` 的额外伤害。
  - 后续再以“原伤害量 + 均衡额外伤害”为基础触发 `绝地`。
  - `均衡` 和 `绝地` 的额外伤害仍可再次吃全增伤类属性、神能压制、回合增伤、石墨限制和攻击减伤。
  - 因此旧公式里 `均衡`、`绝地` 内部再次乘 `(1 + 全伤) * (1 + 极鹿角 * 回合)` 是合理方向，不应简化成只乘 `45%` 或 `18%`。

详见 `knowledge/hero enabling/README.md`。

### 技能段数

截图最后单独乘：

```text
技能段数
```

对多段技能很关键。

阿斯布例子：

- 用户确认：主动技能伤害按一次攻击整体结算；如果攻击过程中跨血量阶段，段数仍按出手时的 Boss 血量阶段计算，不按每段实时血量重新判断。
- `灵魂战栗` 基础是 `2 次 1200%`，技能段数可记为 2。
- 如果目标低血线触发再次造成 `2 次 1200%`，则这部分可能另算一组 2 段。
- `灵枢·星跃` 的 `额外造成 1200% 攻击伤害` 先按独立 1 段记录。
- `同排额外伤害` 的 `1200% 攻击伤害` 先按独立 1 段记录；用户确认打章鱼 Boss 时可以稳定命中。
- `灵枢·月渊` 飞升战技是 `2 次 1500%`，按“一回合总伤害共享均衡”口径，触发当回合可额外计入 2 段。
- `魔王之威` 对 Boss 的“追加 1 次本次技能伤害量 120% 的伤害”可能是额外伤害，不一定等同于技能段数，需要确认。

阿斯布对 Boss 的主动段数口径：

| 场景 | 可计入段数 | 说明 |
| --- | ---: | --- |
| 基础主动 | 2 | `灵魂战栗` 对目标 2 次 1200%。 |
| 目标低血线追伤 | +2 | 基础为低于 40%；主动渊之升华后变为低于 60%。 |
| 灵枢·星跃 | +1 | 目标生命值不低于 50% 时额外 1200%。 |
| 同排额外伤害 | +1 | 本次攻击后对同排敌人额外 1200%；打章鱼 Boss 时稳定命中。 |
| 灵枢·月渊 | +2 | 主动后触发飞升战技，造成 2 次 1500%。 |
| 魔王之威 | 待确认 | 追加本次技能伤害量 120%，先不当作普通独立暴击段。 |

因此计算阿斯布暴击率时，`n` 不应固定写死为 2：

- 只看基础主动：`n = 2`。
- 高血线且有 `灵枢·星跃`，不计月渊：`n = 4`，因为章鱼 Boss 稳定吃到同排额外伤害。
- 再触发 `灵枢·月渊`：通常在上面基础上再 `+2` 段。
- 低血线触发追伤：通常在实际命中的段数上再 `+2` 段。

## 和阿斯布打 Boss 的关系

阿斯布主 C 打 Boss 时，可以把主要伤害拆成：

1. 主动技能 `灵魂战栗` 的基础段数：
   - `2 次 1200%`。
2. 血线追伤：
   - 目标低于 40% 时再 `2 次 1200%`。
   - 有 `渊之升华` 后门槛变为低于 60%。
3. Boss 免恐惧时的 `魔王之威` 补偿：
   - 额外追加 1 次本次技能伤害量 120% 的伤害。
   - 偷取目标 30% 攻击力。
4. 灵枢额外伤害：
   - `灵枢·生辉`：目标最大生命上限 8%，上限不超过阿斯布攻击的 1500%。
   - `灵枢·星跃`：目标生命值不低于 50% 时，额外造成 1200% 攻击伤害。
   - `灵枢·日晕`：自身全伤增加 15%，持续 2 回合。
5. 飞升战技：
   - 基础形态：同排 2000% 攻击伤害。
   - `灵枢·月渊`：全体 2 次 1500% 攻击伤害，并稳定施加深渊凝视和降攻。

公式对阿斯布的关键提示：

- 攻击、神圣伤害、技能伤害率、精准、暴击/暴伤、全伤、鹿角回合成长、技能段数都会直接影响 Boss 输出。
- 如果能量超过 100，溢出能量也会进入技能伤害率乘区。
- 多段技能要拆清楚段数，否则会明显低估或高估输出。
- `魔王之威` 的 Boss 额外伤害按用户补充属于根据伤害量再次造成的真实伤害；它会在基础技能伤害已吃过全增伤类属性后，再次吃全增伤类属性。
- 如果同一回合再触发 `均衡` 和 `绝地`，全增伤类属性会继续在这些伤害量真伤上重复放大，这解释了暗战打 Boss、奈子打 Boss 和均衡炮伤害异常高的来源。
