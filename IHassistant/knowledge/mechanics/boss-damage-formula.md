# 打 Boss 伤害公式

## 来源

- 来源截图：`Weixin Image_20260527034037_7293_38.jpg`
- 补充来源：`【精品】关于全增伤和全减伤的计算关系 by秋叶&爱哈卷 25.12.28.pdf`
- 补充来源：`伤害类型.pdf`
- 用户说明：这是计算打 Boss 伤害的公式。
- 状态：已按截图整理；部分专有名词和乘区来源待确认。

## 场景限制

- 基础公式用于章鱼 Boss / 星域远征 Boss 的传统伤害估算。
- 远征星印、远见套、卡尔/命运/布尔等星印套装加成只适用于星域远征/远征 Boss 场景。
- 如果计算普通 Boss、非远征 PVE 或 PVP，不要默认计入星印相关加成。

## 截图公式整理

对章鱼产生的总输出：

```text
总输出 =
{ 攻击 * [1 - 护甲减伤 * (1 - 护甲减免) * (1 - 破甲)]
  + 攻击 * 神圣伤害 * 70%
}
* (主动技能系数 + 技能伤害率 + 溢出 100 能量后对等计算为技能伤害率)
* (1 + 精准 * 0.3%)
* (1 + 观心印记)
* (暴击 1.5 倍 + 爆伤 * 2)
* (1 + 女皇侵蚀)
* (1 + 魔兽增伤 * 层数)
* (1 + 对持续伤害增伤 * 回合)
* (1 + 全伤)
* (1 + 极鹿角 * 回合)
* [1 + 均衡 45% * (1 + 全伤) * (1 + 极鹿角 * 回合)]
* [1 + 绝地 18% * (1 + 全伤) * (1 + 极鹿角 * 回合)]
* 技能段数
* [1 + (英雄神能差值 + 集体神能差值) / 10]
```

## 与已学机制的对照

### 适用伤害类型

这张 Boss 公式主要适用于 `传统伤害`，也就是基于攻击者自身攻击力、会吃神圣伤害/破甲/精准/暴击/减伤/观心印记等乘区的伤害。

- `真实伤害` 不能直接套完整公式：
  - 通常不吃神圣伤害、破甲、精准、护甲、减伤率、暴击、格挡、观心印记、魔兽增伤等。
  - 通常仍需考虑全伤、全减伤、神能差值、护盾和闪避。
- `惩戒伤害` 更不能套完整公式：
  - 常规攻防属性、全伤、全减伤、护盾等通常都不改变惩戒伤害。
  - 当前按只受神能差值影响记录。

详见 `damage-types.md`。

### 基础伤害

截图中的基础部分：

```text
攻击 * [1 - 护甲减伤 * (1 - 护甲减免) * (1 - 破甲)]
+ 攻击 * 神圣伤害 * 70%
```

和已有 `神圣伤害` 记录基本一致：

```text
基础伤害 =
A 的攻击力 * (1 - D 的护甲减伤率 * (1 - A 的破甲率))
+ A 的攻击力 * A 的神圣伤害率 * 0.7
```

新增差异：

- 新公式多了 `(1 - 护甲减免)` 这一项。
- 用户确认：`护甲减免` 和 `破甲` 暂按同一类护甲抵消/降护甲效果理解；破甲 100% 可以理解为降护甲 100%。
- 记录口径：
  - 后续如果技能描述写“破甲 100%”或“降低护甲 100%”，默认都按抵消目标护甲减伤处理。
  - 若后续发现两个词在游戏内有不同乘区或叠加方式，再单独修正。

### 技能伤害率与能量溢出

截图把以下三项放在同一个加法乘区：

```text
主动技能系数 + 技能伤害率 + 溢出 100 能量后对等计算为技能伤害率
```

这和已有机制能对上：

- 主动技能有自己的倍率，例如阿斯布 `灵魂战栗` 是 `1200%`。
- `增加技能伤害率` 直接加到主动技能伤害百分比中。
- 能量超过 100 时，每溢出 1 点，按 1% 技能伤害率对主动技能加成。

对阿斯布举例：

- 如果一段主动是 `1200%`，阿斯布有 `300% 增加技能伤害率`，且释放时能量为 120，则这一乘区可理解为：
  - `1200% + 300% + 20%`
- 如果技能描述为 `2 次 1200%`，则公式里后面再乘 `技能段数 = 2`。

### 精准

截图：

```text
1 + 精准 * 0.3%
```

和已有机制一致：

- 每 1% 精准增加 0.3% 伤害。
- 旧图记录最高增加 45% 伤害，即精准 150% 到增伤上限。

### 观心印记

公式中的 `观心印记` 来源于观心者（Heart Watcher / HW）的 `观心者印记`。

- 主动技能 `心灵拷问` 施加的印记：
  - 目标被攻击时额外受到 45% 伤害。
- 普通攻击被动 `弱点打击` 施加的印记：
  - 目标被攻击时额外受到 35% 伤害。
- 印记可叠加。
- 主动 45% 和普攻 35% 直接相加，共用 300% 上限。
- 对单一目标最高叠加至 300% 额外受伤。
- 当前默认：`观心者印记` 不会被清除。

当前公式口径：

```text
观心印记 = 目标身上观心者印记累计增伤，最高 300%
```

详见 `knowledge/heroes/HW/heart-watcher.md`。

### 女皇侵蚀

公式中的 `女皇侵蚀` 来源于血腥女王-哈尔拉（Halora / SQH / 女王）的 `深渊侵蚀`。

- 女王主动技能 `深渊降临` 对所有敌人施加 1 层 `深渊侵蚀`。
- `深渊侵蚀` 效果：
  - 被侵蚀英雄暴击伤害降低 24%。
  - 被侵蚀英雄受到的暴击伤害提高 24%。
- 叠加规则：
  - 可以叠加。
  - 每层 24% 直接相加。
  - 上限为增加 200% 伤害。
  - Boss 战斗通常会打满 15 回合，当前默认按最终叠满 `女皇侵蚀 = 200%` 计算。
- 升华强化：
  - `渊之升华`：被侵蚀英雄暴击降低 10%。
  - `崇之升华`：施加 `深渊侵蚀` 时，有 25% 概率额外施加 1 层。

当前公式口径：

```text
女皇侵蚀 = 目标身上深渊侵蚀累计增伤，最高 200%；Boss 战默认按 200%
```

详见 `knowledge/heroes/queen/queen.md`。

### 暴击与暴击伤害

截图：

```text
暴击 1.5 倍 + 爆伤 * 2
```

和已有暴击公式一致：

```text
暴击伤害 = 伤害 * (1.5 + 2 * 暴击伤害率)
```

需要注意：

- 该乘区只在实际暴击时成立。
- 如果没有暴击，理论上应不套用 `1.5 + 爆伤 * 2`，或者按非暴击乘区处理；截图没有写非暴击分支。

#### 暴击率调控口径

如果不考虑 `均衡`，暴击率越高期望伤害越高，因为暴击乘区为：

```text
暴击乘区 = 1.5 + 2 * 暴击伤害率
```

如果带 `均衡`，则暴击和未暴击有分支：

- 暴击：吃暴击乘区，但 `均衡` 转为回血，不提供额外伤害。
- 未暴击：不吃暴击乘区，但触发 `均衡` 额外伤害。
- 用户补充：如果一回合内是多段伤害，只要其中任意一段没有暴击，本次多段伤害里的其他攻击段也会吃到 `均衡` 的增伤。

15 回合、鹿角按极 1 鹿角 `9% * 回合` 计算时：

```text
均衡增伤乘区 B = 1 + 45% * (1 + 全伤) * (1 + 9% * 15)
              = 1 + 1.0575 * (1 + 全伤)
```

暴击是否优于未暴击，取决于：

```text
1.5 + 2 * 暴击伤害率 > 1 + 1.0575 * (1 + 全伤)
暴击伤害率 > 27.875% + 52.875% * 全伤
```

因此单段伤害的默认判断：

- 不带 `均衡`：暴击率尽量高。
- 带 `均衡` 且暴击乘区高于未暴击均衡乘区：暴击率尽量高。
- 带 `均衡` 且未暴击均衡乘区高于暴击乘区：暴击率不宜继续堆高。

常用阈值参考：

| 全伤 | 暴击优于未暴击均衡所需暴击伤害率 |
| ---: | ---: |
| 0% | 27.875% |
| 50% | 54.3125% |
| 100% | 80.75% |
| 150% | 107.1875% |
| 200% | 133.625% |
| 231% 以上 | 超过 150% 暴击伤害上限，未暴击均衡更优 |

单段伤害时，如果不带 `猎手专注`，没有必须卡 `50%` 的硬门槛；理论最优可以低于 50%。如果带 `猎手专注`，且判断未暴击均衡更优，暴击率才建议控制在 `50%` 刚过线的区域，通常记为 `50%~55%`。

多段伤害时，因为任意一段未暴击就能让本次多段伤害吃到 `均衡`，暴击率可以比单段更高。

假设：

- 暴击伤害率满值 `150%`，暴击乘区 `M = 1.5 + 2 * 150% = 4.5`。
- 一次攻击有 `n` 段，每段独立按面板暴击率 `c` 判定。
- 只要任意一段未暴击，本次多段伤害均吃 `均衡增伤乘区 B`。

则每段平均期望乘区可按：

```text
E = c^n * M + c * (1 - c^(n-1)) * M * B + (1 - c) * B
```

最优暴击率近似为：

```text
c* = [B * (M - 1) / (n * M * (B - 1))] ^ [1 / (n - 1)]
```

其中 `M = 4.5`。常用口径：

- `n = 2` 时，`c* = 3.5B / [9 * (B - 1)]`。
- 当 `全伤 = 0%`，`B = 2.0575`，两段最优暴击率约 `75.7%`。
- 当 `全伤 = 100%`，`B = 3.115`，两段最优暴击率约 `57.3%`。
- 当 `全伤 = 200%`，`B = 4.1725`，两段最优暴击率约 `51.1%`。
- 当 `全伤 >= 231%`，两段理论最优会低于 `50%`；不带 `猎手专注` 时可以低于 50%，带 `猎手专注` 时才需要压回 `50%~55%`。
- `n >= 3` 时，应按段数重算，不应简单沿用两段结论。

满爆伤、不带 `猎手专注` 的常用最优暴击率参考：

| 全伤 | 2 段 | 3 段 | 4 段 | 5 段 | 6 段 | 8 段 |
| ---: | ---: | ---: | ---: | ---: | ---: | ---: |
| 0% | 75.7% | 71.0% | 72.3% | 74.2% | 75.9% | 78.8% |
| 50% | 63.4% | 65.0% | 68.2% | 71.0% | 73.3% | 76.9% |
| 100% | 57.3% | 61.8% | 65.9% | 69.2% | 71.8% | 75.8% |
| 150% | 53.6% | 59.8% | 64.5% | 68.0% | 70.9% | 75.0% |
| 200% | 51.1% | 58.4% | 63.5% | 67.3% | 70.2% | 74.5% |
| 231% | 50.0% | 57.7% | 63.0% | 66.9% | 69.9% | 74.3% |
| 300% | 48.1% | 56.6% | 62.2% | 66.2% | 69.3% | 73.9% |

#### 阿斯布初始暴击率倒推

计算阿斯布初始暴击率时，需要把战斗内暴击率来源倒扣出去。

当前已记录的常见来源：

- 凤凰符文强化：全队 `+15% 暴击率`，按常驻处理。
- 阿斯布主动净之升华：`灵魂战栗 +10% 暴击`，只对主动技能判定生效。
- 阿斯布 `致命抓握`：普通攻击后自身 `+24% 暴击率`，持续 3 回合；渊之升华后持续 4 回合。因为持续多回合，计算 Boss 输出时需要考虑可能叠加；如果再次满足触发条件，且前一层加成仍有效，则继续叠加。每层都有自己的持续回合限制，不是永久叠加，也不是只刷新持续时间。叠加 2 层时覆盖值为 `24% * 2 = 48%`。
- 阿斯布 `死亡扩散` 渊之升华：敌方死亡时额外 `+10% 暴击率`，Boss 单体场景不稳定，默认不计入。
- 远征星印 `卡尔 2 件套`：`+10% 暴击率`，只在星域远征/远征 Boss 场景计入；远征常用星印组合里通常会触发。
- 远征星印 `远见 4 件套`：只在星域远征/远征 Boss 场景计入；主动技能后自身 `+30% 暴击率`、`+100% 暴击伤害`，持续 2 回合；用户确认从本次主动结束后开始计算，不作用于本次主动技能的后续段数。

倒推公式：

```text
初始暴击率 = 目标实际暴击率 - 常驻星印暴击率 - 凤凰 15% - 主动净升华 10% - 致命抓握覆盖值 - 其他覆盖中暴击率
```

其中：

- 常用远征星印组合触发 `卡尔 2 件套` 时，`常驻星印暴击率 = 10%`。
- 如果主动回合没有吃到 `致命抓握`，`致命抓握覆盖值 = 0%`。
- 如果主动回合稳定吃到 1 层 `致命抓握`，`致命抓握覆盖值 = 24%`。
- 如果主动回合吃到 2 层叠加的 `致命抓握`，`致命抓握覆盖值 = 48%`。
- `致命抓握` 的每一层都受自己的持续回合限制，计算时需要按实际普攻触发时间线判断还有几层仍然有效。
- 如果只部分覆盖，可按期望覆盖折算，例如覆盖一半时先用 `12%` 粗算。

远征章鱼默认节奏：

- 阿斯布通常是队伍最高攻击目标，会被章鱼 Boss 回合结束的 `深渊虹吸` 扣 100 能量。
- 默认前 1-6 回合阿斯布都释放主动技能，来源是额外星魂系统加成。
- 第 7 回合开始普攻，之后按 `普攻 -> 主动技能` 交替。
- 14 或 15 回合内，主动回合默认为 `1, 2, 3, 4, 5, 6, 8, 10, 12, 14`。第 15 回合是普攻，不额外增加主动次数。
- 如果有 `灵枢·幻黎`，每次主动命中后获得 6 层 `飞升力`；飞升战技 12 层触发并消耗 12 层。
- 因此 14/15 回合内，飞升战技默认在第 `2, 4, 6, 10, 14` 回合触发，共 5 次。
- `致命抓握` 渊之升华后持续 4 回合。
- 第 7 回合第一次普攻后给 1 层，第 8 回合主动通常覆盖 1 层。
- 第 9 回合第二次普攻后进入 2 层覆盖，第 10 回合开始的主动通常覆盖 2 层，也就是 `24% * 2 = 48%`。
- 因此前 1-6 回合主动按 0 层 `致命抓握`，第 8 回合主动按 1 层，第 10 回合及以后主动按 2 层倒推；节奏被打断时再单独修正。

按上述节奏，14/15 回合内阿斯布主动段数：

| Boss 血量阶段 | 无飞升主动 | 触发飞升主动 | 10 次主动总段数 |
| --- | ---: | ---: | ---: |
| 60% 以上 | 4 段 | 6 段 | 50 段 |
| 50%-60% | 6 段 | 8 段 | 70 段 |
| 50% 以下 | 5 段 | 7 段 | 60 段 |

满爆伤、带均衡、无猎手专注时，如果暂不考虑回合内暴击率 Buff 差异，只按 10 次主动的段数结构求目标实际暴击率，参考值如下：

| 全伤 | 60% 以上 | 50%-60% | 50% 以下 |
| ---: | ---: | ---: | ---: |
| 100% | 69.9% | 74.2% | 72.2% |
| 150% | 68.7% | 73.4% | 71.3% |
| 200% | 67.9% | 72.8% | 70.6% |
| 300% | 66.9% | 72.1% | 69.7% |

但是远征常用配置下，实际暴击率 Buff 很高：

- 前 1-6 回合主动：凤凰 15% + 主动净升华 10% + 卡尔 2 件套 10% = 35%，再加初始暴击率。
- 第 8 回合主动：上述 35% + 1 层 `致命抓握` 24% = 59%，再加初始暴击率。
- 第 10 回合及以后主动：上述 35% + 2 层 `致命抓握` 48% = 83%，再加初始暴击率。

因此如果按 14/15 回合整体输出反推初始暴击率，常见全伤区间下最优初始暴击率很低：

| 全伤 | 60% 以上 | 50%-60% | 50% 以下 |
| ---: | ---: | ---: | ---: |
| 100% | 约 5% | 约 6% | 约 5% |
| 150% | 约 4% | 约 5% | 约 4% |
| 200% | 约 3% | 约 4% | 约 4% |
| 300% | 约 2% | 约 3% | 约 3% |

当前实战口径：远征章鱼阿斯布如果使用凤凰、卡尔 2 件套、主动净升华、2 层致命抓握的默认循环，初始暴击率基本不需要额外堆；通常控制在 `0%-5%` 即可。若前 1-6 回合输出权重大、或缺少卡尔 2 件套/凤凰/主动净升华，再重新倒推。

如果按回合数增伤口径，最后两次主动通常占主要输出，前面主动可以近似忽略。此时只看第 12 回合和第 14 回合：

- 第 12 回合：主动，不触发飞升战技。
- 第 14 回合：主动，并触发飞升战技。

| ---: | ---: | ---: | ---: |
| 100% | 70.4% | 74.6% | 72.7% |
| 150% | 69.2% | 73.7% | 71.7% |
| 200% | 68.4% | 73.1% | 71.0% |
| 300% | 67.3% | 72.3% | 70.0% |

如果只看最后一次第 14 回合主动，段数为：

| 全伤 | 60% 以上：6 段 | 50%-60%：8 段 | 50% 以下：7 段 |
| ---: | ---: | ---: | ---: |
| 100% | 72.0% | 75.9% | 74.1% |
| 150% | 71.0% | 75.2% | 73.3% |
| 200% | 70.3% | 74.6% | 72.7% |
| 300% | 69.4% | 74.0% | 71.9% |

远征稳定期已有固定暴击 Buff：

```text
凤凰 15% + 主动净升华 10% + 卡尔 2 件套 10% + 致命抓握 2 层 48% = 83%
```

因此不管按最后两次主动还是只看第 14 回合，远征章鱼阿斯布稳定期都已经超过最佳实际暴击率区间；初始暴击率最优仍然是 `0%`，多出来的初始暴击率基本是溢出属性。

如果印痕灌注选择 `利刃 分支2 右线` 终点：

```text
使自身暴击降低 50%，使自身攻击增加 50%
```

则稳定期固定暴击需要重新倒推。若同时点满 `利刃 分支1 左线` 暴击率线，沿途静态暴击按 `3.3% * 6 = 19.8%` 计入；该线关键节点的 `造成伤害后自身暴击 +8%，持续 2 回合` 只计入已经在当前主动前生效的层数，本次主动造成伤害后获得的 `+8%` 不影响本次主动已经结算的段数。

远征章鱼第 10 回合以后，按凤凰、主动净升华、卡尔 2 件套和 2 层 `致命抓握` 覆盖计算：

```text
基础覆盖 = 凤凰 15% + 主动净升华 10% + 卡尔 2 件套 10% + 致命抓握 48% - 印痕终点 50% + 利刃暴击线静态 19.8%
基础覆盖 = 52.8%

最优初始暴击率 = 目标实际暴击率 - (52.8% + 8% * 利刃暴击线动态层数)
```

按最后两次主动为主要伤害时，参考值如下：

| 利刃暴击线动态层数 | 60% 以上：目标 67.3%-70.4% | 50%-60%：目标 72.3%-74.6% | 50% 以下：目标 70.0%-72.7% |
| ---: | ---: | ---: | ---: |
| 0 层 | 初始约 14.5%-17.6% | 初始约 19.5%-21.8% | 初始约 17.2%-19.9% |
| 1 层 | 初始约 6.5%-9.6% | 初始约 11.5%-13.8% | 初始约 9.2%-11.9% |
| 2 层 | 初始约 0%-1.6% | 初始约 3.5%-5.8% | 初始约 1.2%-3.9% |

实战口径：如果能确认 `利刃 分支1 左线` 的 `+8% 暴击` 在最后两次主动前通常覆盖 1 层，则初始暴击大约控制在 `7%-14%`；如果能稳定覆盖 2 层，则初始暴击大约 `0%-6%`。若该动态暴击层数不稳定，先按 1 层估算更保守。

对满爆伤、带均衡、无猎手专注的阿斯布，常用初始暴击率参考如下。该表已扣除凤凰和主动净升华，未扣除 `卡尔 2 件套`：

| 实际段数 | 目标实际暴击率 | 致命抓握 0 层 | 致命抓握 1 层 | 致命抓握 2 层 |
| ---: | ---: | ---: | ---: | ---: |
| 2 段 | 约 50%~57% | 约 25%~32% | 约 1%~8% | 约 0% |
| 4 段 | 约 63%~66% | 约 38%~41% | 约 14%~17% | 约 0% |
| 6 段 | 约 70%~72% | 约 45%~47% | 约 21%~23% | 约 0% |

因此阿斯布不要只看面板初始暴击率，要看主动释放当下的实际暴击率。若凤凰常驻且主动净升华已点，面板/配置里的初始暴击率通常需要比目标实际暴击率低 `25%`；如果主动阶段稳定覆盖 1 层 `致命抓握`，则需要低 `49%`；如果覆盖 2 层，则需要低 `73%`，很多段数场景下初始暴击率理论上可以接近 0。远征章鱼长线稳定期默认按 2 层 `致命抓握` 倒推。

在星域远征/远征 Boss 常用星印组合下，如果额外触发 `卡尔 2 件套`，上述初始暴击率还要再降低 `10%`。`远见 4 件套` 的主动后 `+30% 暴击率 / +100% 暴击伤害` 不作用于本次主动段数，计算当前主动最佳暴击率时不要扣除这部分。

非远征 Boss 场景不计入 `卡尔 2 件套`、`远见 4 件套` 等星印加成。

### 全伤

截图把 `全伤` 作为单独乘区：

```text
1 + 全伤
```

这说明“造成全伤害增加”类效果与技能系数、精准、暴击等乘区相乘。

新文章补充：

- `全伤` / `全增伤` 与目标的 `全伤害减免` / `全减伤` 不是减法抵消。
- 二者按独立乘区计算：

```text
伤害 = 原始输出 * (1 + 攻击者全伤) * (1 - 目标有效全伤害减免)
```

- 例如攻方全伤 10%、守方有效全伤害减免 6%，最终比例是 `1.10 * 0.94 = 1.034`，不是 `1 + 10% - 6%`。
- 如果攻击者有 `抵消全伤害减免`，则先计算：

```text
目标有效全伤害减免 = max(0, 目标全伤害减免 - 攻击者抵消全伤害减免)
```

- 当前章鱼 Boss 公式截图没有单独列出 Boss 的全伤害减免乘区；后续遇到带全减伤的 Boss 或 PVP 目标时，需要把该乘区补进去。

已整理神器中相关例子：

- 鹿角：每回合结束提高自身造成的全部伤害。
- 绿琴：战斗开始提高自身造成全伤害。
- 镜子：战斗开始以及每过 3 回合提高我方全体造成全伤害。
- 鲨鱼：`狂热推进` 每层提高自身造成全伤害。

### 均衡与绝地

截图中有两个额外乘区：

```text
[1 + 均衡 45% * (1 + 全伤) * (1 + 极鹿角 * 回合)]
[1 + 绝地 18% * (1 + 全伤) * (1 + 极鹿角 * 回合)]
```

这两个乘区比较特殊：

- 它们不是简单的 `1 + 45%` 或 `1 + 18%`。
- 它们内部还再次乘了 `(1 + 全伤)` 和 `(1 + 极鹿角 * 回合)`。
- 这意味着 `均衡`、`绝地` 可能是额外伤害/追加伤害类机制，而不是普通增伤。

- 用户确认：
  - `绝地` 来自赋能 2 第二列。
  - `均衡` 来自赋能 5 第一列。
  - 原公式里的 `绝地 12%` 是未灌注版本；当前默认使用赋能灌注后的算法，按 `18%` 计算。
  - 原公式里的 `均衡 30%` 是未灌注版本；当前默认使用赋能加强版 `lv 灌注`，按 `45%` 计算。
  - 每个赋能选项的 `lv1` 和 `lv2` 子行默认都按已生效处理。

详见 `knowledge/hero enabling/README.md`。

### 技能段数

截图最后单独乘：

```text
技能段数
```

对多段技能很关键。

阿斯布例子：

- 用户确认：主动技能伤害按一次攻击整体结算；如果攻击过程中跨血量阶段，段数仍按出手时的 Boss 血量阶段计算，不按每段实时血量重新判断。
- `灵魂战栗` 基础是 `2 次 1200%`，技能段数可记为 2。
- 如果目标低血线触发再次造成 `2 次 1200%`，则这部分可能另算一组 2 段。
- `灵枢·星跃` 的 `额外造成 1200% 攻击伤害` 先按独立 1 段记录。
- `同排额外伤害` 的 `1200% 攻击伤害` 先按独立 1 段记录；用户确认打章鱼 Boss 时可以稳定命中。
- `灵枢·月渊` 飞升战技是 `2 次 1500%`，如果按“一回合多段伤害共享均衡”口径，触发当回合可额外计入 2 段。
- `魔王之威` 对 Boss 的“追加 1 次本次技能伤害量 120% 的伤害”可能是额外伤害，不一定等同于技能段数，需要确认。

阿斯布对 Boss 的主动段数口径：

| 场景 | 可计入段数 | 说明 |
| --- | ---: | --- |
| 基础主动 | 2 | `灵魂战栗` 对目标 2 次 1200%。 |
| 目标低血线追伤 | +2 | 基础为低于 40%；主动渊之升华后变为低于 60%。 |
| 灵枢·星跃 | +1 | 目标生命值不低于 50% 时额外 1200%。 |
| 同排额外伤害 | +1 | 本次攻击后对同排敌人额外 1200%；打章鱼 Boss 时稳定命中。 |
| 灵枢·月渊 | +2 | 主动后触发飞升战技，造成 2 次 1500%。 |
| 魔王之威 | 待确认 | 追加本次技能伤害量 120%，先不当作普通独立暴击段。 |

因此计算阿斯布暴击率时，`n` 不应固定写死为 2：

- 只看基础主动：`n = 2`。
- 高血线且有 `灵枢·星跃`，不计月渊：`n = 4`，因为章鱼 Boss 稳定吃到同排额外伤害。
- 再触发 `灵枢·月渊`：通常在上面基础上再 `+2` 段。
- 低血线触发追伤：通常在实际命中的段数上再 `+2` 段。

## 和阿斯布打 Boss 的关系

阿斯布主 C 打 Boss 时，可以把主要伤害拆成：

1. 主动技能 `灵魂战栗` 的基础段数：
   - `2 次 1200%`。
2. 血线追伤：
   - 目标低于 40% 时再 `2 次 1200%`。
   - 有 `渊之升华` 后门槛变为低于 60%。
3. Boss 免恐惧时的 `魔王之威` 补偿：
   - 额外追加 1 次本次技能伤害量 120% 的伤害。
   - 偷取目标 30% 攻击力。
4. 灵枢额外伤害：
   - `灵枢·生辉`：目标最大生命上限 8%，上限不超过阿斯布攻击的 1500%。
   - `灵枢·星跃`：目标生命值不低于 50% 时，额外造成 1200% 攻击伤害。
   - `灵枢·日晕`：自身全伤增加 15%，持续 2 回合。
5. 飞升战技：
   - 基础形态：同排 2000% 攻击伤害。
   - `灵枢·月渊`：全体 2 次 1500% 攻击伤害，并稳定施加深渊凝视和降攻。

公式对阿斯布的关键提示：

- 攻击、神圣伤害、技能伤害率、精准、暴击/暴伤、全伤、鹿角回合成长、技能段数都会直接影响 Boss 输出。
- 如果能量超过 100，溢出能量也会进入技能伤害率乘区。
- 多段技能要拆清楚段数，否则会明显低估或高估输出。
