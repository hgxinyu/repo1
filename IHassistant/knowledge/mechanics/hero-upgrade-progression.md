## 阶段顺序

普通 E5 英雄继续升级时，当前按以下阶段理解：

1. E5 -> E9。
2. E9 -> 根源 120。
3. 补齐核武和技能点门槛。
4. 根源 120 -> 飞升等级 1-6，也就是神能展示中的神一到神六基础。

## E5 到 E9

当前资料库中的 E9 蓝碎口径可由飞升殿堂 1 级前置条件和根源 120 资源交叉校验：

```text
E9 蓝碎 = 飞升殿堂 1 级前置星碎 - 根源 120 印痕
E9 蓝碎 = 14,929,100 - 9,994,100
E9 蓝碎 = 4,935,000
```

因此，一个普通 E5 英雄升到 E9，当前记录为需要 `4,935,000` 星碎 / 蓝碎 / 印痕。

## E9 到根源 120

从 E9 升到 120 级满根源，需要：

- 印痕：`9,994,100`
- 意识精华：`1,248,120`
- 核武箱子：`8` 个。
  - 每个箱子 `50` 个碎片。
  - 合计 `400` 个核武碎片。
- 技能点：`16/720K`。
  - 主动：`4` 点。
  - 被动 1：`4` 点。
  - 被动 2：`4` 点。
  - 被动 3：`4` 点。

把 E5 -> E9 的蓝碎和根源 120 累计印痕合并后：

```text
4,935,000 + 9,994,100 = 14,929,100
```

这与飞升殿堂 1 级的 `前置星碎 = 14,929,100` 对上；`前置意识 = 1,248,120` 也与根源 120 意识精华对上。

注意：飞升殿堂截图的前置列只列出星碎和意识精华。核武箱子和技能点是用户补充的额外进入飞升门槛，应和前置星碎 / 前置意识一起检查。

## 总口径

普通 E5 英雄一路升到神六，当前资源链路可拆成：

```text
E5 -> E9：4,935,000 蓝碎
E9 -> 根源 120：9,994,100 印痕 + 1,248,120 意识精华 + 8 个核武箱子（400 碎片）+ 技能点 16/720K（主动/被动1/被动2/被动3 各 4 点）
根源 120 -> 神六：60 光玉 + 2,400,536 灵碎 + 5,030,000 时空结晶 + 10,000,490 星辰碎片
```

如果只按星碎 / 蓝碎 / 印痕同资源口径合并：

```text
E5 -> 根源 120 前置星碎 = 14,929,100
根源 120 -> 神六星辰碎片 = 10,000,490
```

注意：`星碎 / 蓝碎 / 印痕`、`星辰碎片` 在不同截图中可能使用不同中文名，但当前按同类 Stellar Shards 资源处理；若后续游戏内界面区分为不同道具，需要重新拆分。

注意：核武和技能点当前按用户补充事实记录，未出现在 `genyuan.jpeg` 和飞升殿堂截图的资源列中。
