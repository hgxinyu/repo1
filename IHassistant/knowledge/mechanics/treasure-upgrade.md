## 事实

原图标题为“珍宝升级碎片统计”，按珍宝颜色、升星阶段和小等级列出碎片消耗。1 星本身也需要碎片，`0星 -> 1星` 没有小阶段，直接消耗与 `1星 -> 2星` 合计相同的碎片。原图里的“升二星”表示当前是 1 星、正在升到 2 星；“升五星”表示当前是 4 星、正在升到 5 星；`4星 -> 5星` 到 `10` 级就是五星满级。

颜色分组：

- 粉橙：粉色 / 橙色珍宝共用同一组消耗。
- 红：红色珍宝。
- 绿：绿色珍宝。

阶段最高小等级：

- `0星 -> 1星`：无小阶段，直接消耗一笔碎片。
- `1星 -> 2星`（原图：升二星）：0 -> 3。
- `2星 -> 3星`（原图：升三星）：0 -> 5。
- `3星 -> 4星`（原图：升四星）：0 -> 7。
- `4星 -> 5星`（原图：升五星）：0 -> 10。
