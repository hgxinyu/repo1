# 神器知识库

这里记录神器截图、等级、简称、属性和效果。

## 等级体系

神器有两个大级别：

- 辉煌神器
- 极境辉煌神器

每个大级别内有三个小级别：

| 小级别 | 名称 |
| --- | --- |
| 1 | 闪烁 |
| 2 | 光辉 |
| 3 | 璀璨 |

## 常用叫法

- 辉煌神器璀璨：通常简称“粉三”。
- 极境辉煌神器璀璨：通常简称“极3”。

## 记录要求

- 保留用户上传截图的原始文件名。
- 从截图整理属性时，区分基础属性、特殊效果、触发条件和待确认内容。
- 当用户使用简称时，按本文件的简称规则理解；遇到未记录简称时先标记待确认。
- 同一神器有多张截图时，按文件名中的 `1`、`2` 等序号合并整理为同一神器档案。
