# 魔兽知识库

每个队伍可以携带 1 个魔兽。魔兽提供三类价值：

- 技能：战斗中释放，造成伤害、治疗、护盾、增伤或减益。
- 光环：直接给队伍提供属性。
- 符文强化：进一步给队伍提供固定属性或百分比属性。

## 释放规则

- 魔兽起始能量为 0。
- 每次任意英雄释放大招/主动技能时，魔兽能量 +10。
- 每过 1 回合，魔兽能量 +20。
- 魔兽能量满 100 时，释放魔兽技能。

## 记录要求

- 每个魔兽通常有 4 张截图：前两张合并为技能和灵契释放强化，后两张分别记录光环和符文强化。
- 阵容建议中需要计入魔兽提供的队伍速度、攻击、生命、减伤、免控、神圣伤害、暴击等属性。
- 涉及 Boss 伤害公式时，魔兽造成的伤害、魔兽增伤层数、对燃烧目标增伤等需要和具体公式分开确认。凤凰的对燃烧目标伤害加成，当前按目标身上有任意来源燃烧效果时，对任意来源伤害都有效处理。
