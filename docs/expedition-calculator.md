# 远征积分计算器

页面：`flipgame/ExpeditionCalculator.html`  
访问权限：VIP 专属。页面加载时会调用 `/api/me` 检查当前登录账号是否具备 `vip` 或管理员权限。  
页面数据：`flipgame/seboss_all.json`  
原始数据：`flipgame/SE Boss New.xlsx`

## 数据口径

Excel 原始数据不是万亿单位。导出到 `seboss_all.json` 时，页面数据统一按万亿基准存储：

```text
json.total = Excel 积分 / 1,000,000,000,000
json.per1 = Excel 1%积分 / 1,000,000,000,000
```

Excel 中可能存在科学计数法，导出时需要按数值处理，避免把科学计数字符串截断或当文本显示。

## 显示规则

- `1 ~ 179` 层：显示万亿单位。
- `180 ~ 200` 层：显示原始分数，不显示万亿单位。
- 默认选择 `100` 层。

## 输入输出

选择层数后显示：

- 总积分
- 1% 血量积分
- 剩余积分

输入剩余血量百分比后：

```text
剩余积分 = 当前层总积分 * 剩余血量百分比 / 100
```

## 公会总分估算

公会总分输入单位为万亿。页面从 200 层开始逐层累计，估算当前所在层。

输出：

- 所在层数：显示 `约`
- 该层剩余进度：显示剩余百分比，不是已打掉百分比

提示文案：

```text
根据手动收集的数据估算，可能会有误差。
```

## 更新数据流程

1. 更新 `SE Boss New.xlsx`。
2. 重新生成 `seboss_all.json`。
3. 通过本地 server 打开页面抽查 100、180、200 等层。
4. 确认 180 以上显示原始分数，179 以下显示万亿。
