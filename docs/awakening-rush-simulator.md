# 觉醒冲榜模拟器

页面：`flipgame/AwakeningRushSimulator.html`

价格数据：`flipgame/quality_prices.json`，直接维护 JSON。

访问权限：VIP 专属。页面加载时会调用 `/api/me` 检查当前登录账号是否具备 `vip` 或管理员权限。

## 规则前提

- 每次觉醒消耗 `100` 绑钻。
- 每次觉醒先计入冲榜分数。
- 觉醒后，`B+` 及以下资质全部分解并返还绑钻。
- `A-`、`A`、`A+`、`S`、`SS`、`SSS` 资质保留，不分解、不返钻。
- 返还的绑钻继续用于后续觉醒。
- 绑钻不足 `100` 时停止模拟。
- 页面输入绑钻数量，最大值为 `2,000,000`。

## 期望值

单次觉醒期望分数：

```text
sum(概率 * 对应分数) = 5.92963
```

单次觉醒期望返钻：

```text
sum(概率 * 分解得钻) = 47.731
```

返钻率：

```text
47.731 / 100 = 0.47731
```

有效觉醒次数：

```text
初始绑钻 / (100 - 47.731)
```

理论期望总分：

```text
有效觉醒次数 * 5.92963
```

理论期望 A 数量：

```text
有效觉醒次数 * (0.00015 + 0.000065 + 0.000025)
= 有效觉醒次数 * 0.00024
```

理论期望 S 数量：

```text
有效觉醒次数 * (0.000005 + 0.000005 + 0.000005)
= 有效觉醒次数 * 0.000015
```

理论期望净投入：

```text
期望投入星钻 = 初始绑钻 / 星钻绑钻比例
期望出货星钻 = 有效觉醒次数 * sum(A-以上各资质概率 * 对应出货资质价格)
期望净投入 = 期望投入星钻 - 期望出货星钻
```

示例：`50,000` 绑钻。

```text
有效觉醒次数 = 50000 / (100 - 47.731) = 956.589948
理论期望总分 = 956.589948 * 5.92963 = 5672.224454
理论期望 A 数量 = 956.589948 * 0.00024 = 0.229582
理论期望 S 数量 = 956.589948 * 0.000015 = 0.014349
如果星钻绑钻比例为 5，且使用当前 quality_prices.json：
单次觉醒期望出货星钻 = 0.00015*6000 + 0.000065*10000 + 0.000025*15000 + 0.000005*50000 + 0.000005*100000 + 0.000005*200000 = 3.675
期望投入星钻 = 50000 / 5 = 10000
期望出货星钻 = 956.589948 * 3.675 = 3515.468059
期望净投入 = 10000 - 3515.468059 = 6484.531941
```

## 随机模拟输出

页面的随机模拟按逐抽流程执行：

```text
while 当前绑钻 >= 100:
  当前绑钻 -= 100
  随机抽取资质
  总分 += 资质分数
  if 资质 <= B+:
    当前绑钻 += 分解得钻
  else if 资质为 A- / A / A+:
    A 数量 += 1
  else:
    S 数量 += 1
```

页面把本次模拟和理论期望放在同一个对照区：

- 总分 / 期望总分。
- A 数量 / 期望 A 数量：`A-`、`A`、`A+` 数量合计。
- S 数量 / 期望 S 数量：`S`、`SS`、`SSS` 数量合计。
- 净投入 / 期望净投入。
- 觉醒次数 / 期望觉醒次数。

辅助输出：

- `A-`、`A`、`A+`、`S`、`SS`、`SSS` 分档数量。

## 净投入

页面从 `quality_prices.json` 读取：

- `starDiamondBoundDiamondRatio`：星钻绑钻比例。
- `keptPrice`：出货资质价格，用于 A- 以上出货折算额外收益。

公式：

```text
投入星钻 = 初始绑钻 / 星钻绑钻比例
出货星钻 = sum(A-以上各资质出货数量 * 对应出货资质价格)
净投入 = 投入星钻 - 出货星钻
```

理论期望对照里的期望净投入使用同一份价格数据，但把实际出货数量替换为：

```text
期望出货数量 = 有效觉醒次数 * 对应资质概率
```

示例：如果输入 `50,000` 绑钻，星钻绑钻比例为 `5`，本次模拟出 `1` 个 `A-`，且 `A-` 出货资质价格为 `6000`：

```text
投入星钻 = 50000 / 5 = 10000
出货星钻 = 1 * 6000 = 6000
净投入 = 10000 - 6000 = 4000
```
