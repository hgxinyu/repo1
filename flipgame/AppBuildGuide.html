建APP帮助（PWA安装指南）

按你的设备选择对应步骤，把工具站添加到主屏或桌面，像 APP 一样直接打开。

iOS（Safari）

  1. 用 Safari 打开网站主页（不是微信内置浏览器）。
  2. 点击底部“分享”按钮（方框+向上箭头）。
  3. 下拉找到并点击“添加到主屏幕”。
  4. 确认名称后点“添加”。
  5. 回到桌面，点击图标即可 APP 模式打开。
如果看不到“添加到主屏幕”，先确认：使用的是 Safari；网址是 HTTPS；页面已完整加载。

Android（Chrome）

  1. 用 Chrome 打开网站主页。
  2. 点右上角菜单（三个点）。
  3. 选择“安装应用”或“添加到主屏幕”。
  4. 确认后安装。
  5. 从桌面或应用列表打开，即可全屏/独立窗口使用。
不同品牌手机文案可能是“Install app / Add to Home screen / 安装”。

Windows（Chrome / Edge）

  1. 在 Chrome 或 Edge 打开网站主页。
  2. 地址栏右侧点“安装应用”图标，或菜单里选“安装此站点为应用”。
  3. 确认安装。
  4. 之后可从开始菜单、桌面快捷方式打开。
  5. 需要卸载时，在浏览器“应用”管理中删除即可。
如果图标没出现，先刷新页面一次，或确认站点不是本地 http:// 地址。

常见问题

  1. 安装后还是浏览器标签页：先删掉旧快捷方式，再重新安装。
  2. 页面更新不及时：强制刷新一次（iOS 可关闭后重开；桌面按 Ctrl+F5）。
  3. 按钮挡住内容：APP 模式会自动启用简化导航和滑动返回主页。
